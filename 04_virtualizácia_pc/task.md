# Porovnanie virtualizačných nástrojov

| Nástroj     | Cena                     | Platformy                           | Výhody                                                                 | Nevýhody                                                               |
|-------------|---------------------------|--------------------------------------|-------------------------------------------------------------------------|------------------------------------------------------------------------|
| VirtualBox  | Zadarmo (open-source)     | Windows, Linux, macOS                | Jednoduchý, zdarma, široká kompatibilita                               | Slabší výkon oproti VMware/Hyper-V, menej pokročilé funkcie           |
| VMware Workstation / Player | Player = zadarmo, Workstation = platený | Windows, Linux | Vysoký výkon, stabilita, veľa funkcií, podpora 3D akcelerácie           | Platená verzia drahá, macOS nepodporuje (okrem Fusion pre macOS)      |
| Hyper-V     | Súčasť Windows Pro/Enterprise (zadarmo v systéme) | Windows | Výborný výkon, „native“ integrácia vo Windows, vhodné pre servery      | Funguje iba na Windows, slabšia kompatibilita niektorých OS           |
